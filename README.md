# Arc Raiders – Ultimate Companion

Arc Raiders – Ultimate Companion is a community-made companion app designed to help players stay informed and prepared in the world of ARC Raiders.

Track live rotating events, explore maps and loot locations, follow quests, and stay up to date with the latest ARC Raiders news — all in one clean and fast companion app.

---

## Features
- Live rotating events and timers
- Maps and loot locations
- Loot database
- Quests and task tracking
- Latest ARC Raiders news and updates
- ARC Hub with essential companion tools
